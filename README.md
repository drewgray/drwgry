# drwgry

Personal portfolio website with interactive particle animation.

## Editing Content

Open `index.html` and look for comments marked `CUSTOMIZE`:

1. **Name** (line ~67): Replace "Drew Gregory" with your name
2. **Tagline** (line ~70): Replace "Software Engineer" with your role
3. **Links** (lines ~73-77): Update the href attributes and link text

## Local Preview

Open `index.html` directly in your browser, or run a local server:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`

## Deploy to GitHub Pages

1. Push this repo to GitHub
2. Go to Settings → Pages
3. Set Source to "Deploy from a branch"
4. Select `main` (or `master`) branch and `/ (root)` folder
5. Save — your site will be live at `https://yourusername.github.io/reponame`

## Deploy to Netlify

1. Push to GitHub/GitLab/Bitbucket
2. Log in to [netlify.com](https://netlify.com)
3. Click "Add new site" → "Import an existing project"
4. Select your repo — no build settings needed
5. Deploy

## Deploy to Vercel

1. Push to GitHub/GitLab/Bitbucket
2. Log in to [vercel.com](https://vercel.com)
3. Click "Add New" → "Project"
4. Import your repo — no configuration needed
5. Deploy
